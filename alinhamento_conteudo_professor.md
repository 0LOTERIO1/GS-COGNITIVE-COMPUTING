# Alinhamento com o conteúdo do professor — Cognitive Computing

Este projeto foi ajustado para deixar explícita a relação com os materiais de aula do professor Arnaldo:

## Material de DataFrame / Pandas
- Carregamento de base CSV com `pd.read_csv()`.
- Visualização inicial com `head()`.
- Verificação de tamanho com `shape`.
- Verificação de tipos com `dtypes` e `info()`.
- Verificação de valores faltantes com `isnull().sum()`.
- Resumo estatístico com `describe()`.
- Agrupamentos com `groupby()`.
- Gráficos exploratórios com `matplotlib`.
- Correlação entre atributos com `corr()`.

## Material de Machine Learning
- Definição de variáveis de entrada `X` e variável alvo `y`.
- Separação treino/teste com `train_test_split()`.
- Uso de codificação One Hot Encoder para variáveis categóricas.
- Treinamento de classificador supervisionado.
- Avaliação com accuracy, precision, recall, f1-score e ROC-AUC.
- Matriz de confusão.
- Interpretação das variáveis mais importantes do modelo.

## GS — Cognitive Computing, Computer Vision and IoT Systems
- Caminho escolhido: Ciência de Dados / Machine Learning.
- Problema conectado ao tema Space Connect.
- Base híbrida incluída (focos reais INPE + variáveis estimadas).
- Análise exploratória com gráficos, correlações e interpretação de variáveis.
- Notebook e script Python reprodutíveis.
- Relatório curto com problema, metodologia, resultados e conclusão.
